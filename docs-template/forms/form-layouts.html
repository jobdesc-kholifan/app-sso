Form Layouts

Responsive and organized form structures for various interface needs.

Form Grid

Responsive multi-column layouts using the built-in grid system.

HTML Example
<div class="form-grid"> <div class="space-y-2"> <label class="form-label">First Name</label> <input type="text" class="form-control"> </div> <div class="space-y-2 lg:col-span-2"> <label class="form-label">Address</label> <input type="text" class="form-control"> </div> </div>
Horizontal Form

Aligned labels and controls on the same row for administrative interfaces.

HTML Example
<div class="form-horizontal"> <div class="form-group"> <label class="form-label md:text-right">Full Name</label> <input type="text" class="form-control" placeholder="Enter name"> </div> </div>
Inline Form

Compact controls on a single row, perfect for filters and toolbars.

HTML Example
<div class="form-inline bg-slate-50 dark:bg-slate-900/50 p-6 rounded-xl border border-dashed"> <input type="text" class="form-control w-auto" style="min-width: 200px;" placeholder="Search files..."> <select class="form-select w-auto" style="min-width: 150px;"> <option>Latest</option> <option>Oldest</option> </select> <button class="btn btn-primary">Apply Filters</button> </div>
