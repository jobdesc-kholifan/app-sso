Components / Tabs

Tabs

Interactive tab system with support for different variants, colors, and sizes. Perfect for organizing content into logical groups.

Content Construction

To make tabs functional, link each .nav-link to its corresponding target using the data-tab-target attribute. The content panes must have the .tab-pane class.

1. The Trigger (Navigation)
Trigger Link

Use attribute: data-tab-target="#content-id"

2. The Target (Content)
Content Pane

Visible when linked trigger is clicked.

Use id: id="content-id"

HTML Example Structure
<!-- 1. Navigation Container --> <div class="nav-tabs"> <div class="nav-link active" data-tab-target="#unique-id">Tab Title</div> </div> <!-- 2. Content Container --> <div class="tab-content"> <div id="unique-id" class="tab-pane">... content here ...</div> </div>
