Components / Modals

Modals

Flexible and highly customizable dialog prompts with support for various sizes and states.

Basic Modals

Standard modals for general purpose information and interaction.

Standard Modal Title

This is vertical-aligned by default. You can put any content here including forms or complex layouts.

Centered Modal

This modal is explicitly centered in the viewport using the modal-center class.

HTML Example
<div class="vibe-modal" id="modal-id"> <div class="vibe-modal-dialog"> <div class="vibe-modal-content"> <div class="vibe-modal-header"> <h5 class="text-lg font-bold">Title</h5> <button onclick="closeModal('modal-id')"><i class="bx bx-x"></i></button> </div> <div class="vibe-modal-body"> <p>Content goes here...</p> </div> <div class="vibe-modal-footer"> <button class="btn btn-ghost">Cancel</button> <button class="btn btn-primary">Save</button> </div> </div> </div> </div>

Scrollable Body

HTML — Long Content (Full Page Scroll)
<!-- Default: modal container scrolls with the page --> <div class="vibe-modal" id="modal-id"> <div class="vibe-modal-dialog"> <div class="vibe-modal-content"> <div class="vibe-modal-header">...</div> <div class="vibe-modal-body"> <!-- Long content here — entire dialog grows and scrolls --> </div> <div class="vibe-modal-footer">...</div> </div> </div> </div>
HTML — Scrollable Body (Fixed Header & Footer)
<!-- Add modal-dialog-scrollable to the dialog wrapper --> <div class="vibe-modal" id="modal-id"> <div class="vibe-modal-dialog modal-dialog-scrollable"> <div class="vibe-modal-content"> <div class="vibe-modal-header">Fixed Header</div> <div class="vibe-modal-body"> <!-- Only this area scrolls; header & footer stay fixed --> </div> <div class="vibe-modal-footer">Fixed Footer</div> </div> </div> </div>
HTML — Sticky Header
<!-- Apply modal-sticky to remove default padding, then use sticky on the header --> <div class="vibe-modal modal-sticky" id="modal-id"> <div class="vibe-modal-dialog"> <div class="vibe-modal-content"> <!-- sticky top-0 keeps header pinned to viewport during scroll --> <div class="vibe-modal-header sticky top-0 z-10 bg-white dark:bg-slate-900 shadow-sm"> Sticky Header Title </div> <div class="vibe-modal-body"> <!-- Long content scrolls past the pinned header --> </div> <div class="vibe-modal-footer">...</div> </div> </div> </div>
