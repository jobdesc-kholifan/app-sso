Components / Buttons

Buttons

Use Vibe's custom button styles for actions in forms, dialogs, and more with support for multiple sizes, states, and more.

Semantic Colors

Standard colors for different semantic meanings.

HTML Example
<button class="btn btn-default">Default</button> <button class="btn btn-primary">Primary</button> <button class="btn btn-info">Info</button> <button class="btn btn-success">Success</button> <button class="btn btn-warning">Warning</button> <button class="btn btn-danger">Danger</button>
Soft Variations

Subtle, modern button styles with low-opacity backgrounds.

HTML Example
<button class="btn btn-soft-primary">Soft Primary</button> <button class="btn btn-soft-info">Soft Info</button> <button class="btn btn-soft-success">Soft Success</button> <button class="btn btn-soft-warning">Soft Warning</button> <button class="btn btn-soft-danger">Soft Danger</button>
Gradient Variations

Rich, vibrant gradients for high-impact call to actions.

HTML Example

With Icons

Combine text with Boxicons for better visual hierarchy.

* Tip: For icon-only buttons, always use the .sr-only utility for accessibility.

HTML Example
<!-- Prefix Icon --> <button class="btn btn-primary"> <i class="bx bx-plus"></i> Add Item </button> <!-- Suffix Icon --> <button class="btn btn-outline btn-primary"> Next Step <i class="bx bx-chevron-right"></i> </button> <!-- Icon Only (Accessible) --> <button class="btn btn-default aspect-square p-2"> <i class="bx bx-dots-horizontal-rounded"></i> <span class="sr-only">Open menu</span> </button>
