Components / Alerts

Alerts

Provide contextual feedback messages for typical user actions with a handful of available and flexible alert messages.

Soft Variations

Subtle backgrounds with distinct semantic color states.

Primary! This is a standard informative alert message.
Success! Your transaction was completed successfully.
Error! Something went wrong with your request.
Warning! Please review your profile before proceeding.
HTML Example
<div class="alert alert-primary"> <i class="bx bxs-info-circle text-xl"></i> <div><strong class="font-bold">Primary!</strong> Message here.</div> </div> <div class="alert alert-success"> <i class="bx bxs-check-circle text-xl"></i> <div><strong class="font-bold">Success!</strong> Completed successfully!</div> </div> <div class="alert alert-danger"> <i class="bx bxs-error text-xl"></i> <div><strong class="font-bold">Error!</strong> Something went wrong.</div> </div> <div class="alert alert-warning"> <i class="bx bxs-error-circle text-xl"></i> <div><strong class="font-bold">Warning!</strong> Please review.</div> </div>
Solid Variations
New Update! Version 2.0 is now available for download.

<div class="alert alert-solid-primary"> <i class="bx bxs-megaphone text-xl"></i> <div><strong class="font-bold">Solid!</strong> This is a solid alert.</div> </div> <div class="alert alert-solid-success"> <i class="bx bxs-party text-xl"></i> <div><strong class="font-bold">Congratulations!</strong> Successful action.</div> </div> <div class="alert alert-solid-danger"> <i class="bx bxs-shield-x text-xl"></i> <div><strong class="font-bold">Danger!</strong> Critical security alert!</div> </div>
